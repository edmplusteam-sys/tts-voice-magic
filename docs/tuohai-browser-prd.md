# 拓海浏览器需求文档（整合版）

## 一、产品定位

跨境电商多店铺防关联浏览器：

**安全多开 + 网络检测 + AI工具 + 多店管理 + 商品AI生成 + 数据化选品**

目标是帮助跨境卖家实现合规、稳定的多店运营，降低封店风险。

---

## 二、核心功能

### 1. 防关联多开

| 功能 | 说明 |
| --- | --- |
| 多店铺独立环境 | 每个店铺拥有独立浏览器环境 |
| 指纹伪装 | UA、时区、语言、硬件信息伪装 |
| Cookie、缓存完全隔离 | 店铺间数据互不干扰 |
| 一店一IP绑定 | 每个店铺绑定独立IP |

### 2. 多店一键管理

| 功能 | 说明 |
| --- | --- |
| 店铺分组 | 按平台/类目分组管理 |
| 批量启动/关闭 | 一键批量操作多个店铺 |
| 一键切换店铺 | 快速切换不同店铺环境 |
| 环境批量复制 | 快速复制店铺配置 |

### 3. 网络环境检测

| 功能 | 说明 |
| --- | --- |
| IP归属地检测 | 检测IP所在国家/地区 |
| IP风险检测 | 检测IP是否被平台标记 |
| 延迟/丢包检测 | 网络质量检测 |
| 时区匹配检测 | 确保时区与IP所在地一致 |
| 平台连通性检测 | 检测与电商平台连接状态 |
| 安全报告 | 安全/警告/危险三级报告 |

### 4. 代理集成（HTTP/Socks5）

| 功能 | 说明 |
| --- | --- |
| 支持代理配置 | 支持HTTP/Socks5协议 |
| 内置电商代理入口 | 集成优质代理服务商 |
| 用户自助购买 | 直接在软件内购买代理 |
| 一键导入代理 | 快速导入代理配置 |

### 5. AI电商工具

| 功能 | 说明 |
| --- | --- |
| AI店铺数据分析 | 分析店铺运营数据 |
| AI商品文案生成 | 自动生成商品标题、描述 |
| AI选品建议 | 基于数据提供选品建议 |
| AI客服话术 | 生成客服回复话术 |

### 6. 商品AI生成

| 功能 | 说明 |
| --- | --- |
| 竞品链接读取 | 输入竞品链接，读取公开信息 |
| AI分析卖点/关键词 | 自动提取产品卖点和关键词 |
| AI生成原创内容 | 生成原创标题、描述、属性 |
| 快捷回填 | 一键回填到自己店铺 |

### 7. 数据化选品工具（新增）

| --- | --- | --- |
| **趋势数据展示** | 销量趋势图 | 显示产品近期订单量变化曲线 |
|  | AliScore评分 | 产品综合评分系统 |
|  | 增长率（Growth） | 显示产品销量增长百分比 |
|  | 竞争度（Competition） | 显示产品竞争激烈程度 |
| **产品信息展示** | 近期订单数 | Recent Orders 数据 |
|  | 总订单数 | Total Orders 数据 |
|  | 价格区间 | 产品销售价格范围 |
|  | 主要销售国家 | Top Country 分析 |
| **数据筛选维度** | 国家分布分析 | 分析不同国家销售情况 |
|  | 利润区间评估 | 显示产品利润范围 |
|  | 起量阶段判断 | 判断产品是否刚起量 |
